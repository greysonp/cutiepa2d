registerCollisionGroup

(
  • name
  • [props]
)
public

Defined in registerCollisionGroup:147

Use this function to register a new Collision Group with the current scene.

Parameters:

  • name String

    A unique name identifying a collision group for this scene.

  • [props] Object optional

    The properties being passed in.

    • [collidesWith] Object optional

      An object defining this collision group's interactions with other collision groups.

      • [name="*"] String optional
        An array of the names of groups which interact with this collision group. "*" indicates all registered groups.
      • [handler] Callback optional
        A collision handler to be called when an object of this group collides with an object in the group specified in the name array. It will have the signature (obj1, obj2, rect). obj1 is the first object involved in the collision. obj2 is the second object involved in the collision. rect is the rectangular area representing the intersection between the objects.
