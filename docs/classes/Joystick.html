Joystick Class

Defined in: Joystick:6
Module: cutie

Creates an on-screen joystick that you can poll for angle and magnitude.

Constructor

Joystick

(
  • [props]
)

Defined in Joystick:6

Parameters:

  • [props] Object optional

    The properties being passed in.

    • [position] Number optional

      The position of the center of the joystick.

      • [x] Number optional
        The x-coordinate of the center of the joystick.
      • [y] Number optional
        The y-coordinate of the center of the joystick.
    • [baseDisk] Number optional

      Properties of the lower base disk of the joystick - the part that doens't move.

      • [color="#ccc"] String optional
        What color the base disk should be.
      • [radius=60] Number optional
        The radius of the base disk.
      • [alpha=0.3] Number optional
        The level of transparency of the base disk.
    • [pointerDisk] Number optional

      Properties of the upper pointer disk of the joystick - the part that moves.

      • [color="#aaa"] String optional
        What color the pointer disk should be.
      • [radius=20] Number optional
        The radius of the pointer disk.
      • [alpha=0.3] Number optional
        The level of transparency of the pointer disk.
