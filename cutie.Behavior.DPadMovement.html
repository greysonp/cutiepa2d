Class: DPadMovement

cutie.Behavior. DPadMovement

new DPadMovement(props)

Gives an object analog movement via an on-screen joystick.
Parameters:
Name Type Argument Description
props Object <optional>
 The properties being passed in.
Properties
Name Type Argument Default Description
speed Number <optional>
 100 The maximum speed the object can travel.
faceDirection Boolean <optional>
 true Whether or not you want the object to face in the direction it's moving.
position Number <optional>
 The position of the center of the DPad.
Properties
Name Type Argument Description
x Number <optional>
 The x-coordinate of the center of the Dpad. By Default it is placed in the bottom left corner.
y Number <optional>
 The y-coordinate of the center of the DPad. By default it is placed in the bottom left corner.
buttonColor String <optional>
 '#ccc' The hex RBG value of the color of the button.
normalAlpha Number <optional>
 1 The unpressed transparency of the buttons.
pressedAlpha Number <optional>
 0.5 The pressed transparency of the buttons.
