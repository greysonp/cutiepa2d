Class: Follow

cutie.Behavior. Follow

new Follow(props)

Gives an object the ability to follow another object or the mouse.
Parameters:
Name Type Argument Description
props Object <optional>
 The properties being passed in.
Properties
Name Type Argument Default Description
targetObj Boolean <optional>
 false This is the object that is going to be followed. If no object is passed in, it will follow the mouse.
speed Number <optional>
 100 This is an integer value giving the speed in px/s
distance Number <optional>
 0 This is the follow distance in px.
setCenter Boolean <optional>
 true This is signifies if the object doing the following should rotate around its center
angleOffset Number <optional>
 90 This is the following orientation where 90 is straight up, the zero point is the cartesian positive x axis and degrees rotate counter-clockwise up from this x axis.
