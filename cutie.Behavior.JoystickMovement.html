Class: JoystickMovement

cutie.Behavior. JoystickMovement

new JoystickMovement(props)

Gives an object analog movement via an on-screen joystick.
Parameters:
Name Type Argument Description
props Object <optional>
 The properties being passed in.
Properties
Name Type Argument Default Description
speed Number <optional>
 5 The maximum speed the object can travel.
faceDirection Boolean <optional>
 false Whether or not you want the object to face in the direction it's moving/
angleOffset Number <optional>
 90 This is the following orientation where 90 is straight up, the zero point is the cartesian positive x axis and degrees rotate counter-clockwise up from this x axis.
position Number <optional>
 The position of the center of the joystick.
Properties
Name Type Argument Description
x Number <optional>
 The x-coordinate of the center of the joystick.
y Number <optional>
 The y-coordinate of the center of the joystick.
baseDisk Number <optional>
 Properties of the lower base disk of the joystick - the part that doens't move.
Properties
Name Type Argument Default Description
color String <optional>
 "#ccc" What color the base disk should be.
radius Number <optional>
 60 The radius of the base disk.

 0.3 The level of transparency of the base disk.
pointerDisk Number <optional>
 Properties of the upper pointer disk of the joystick - the part that moves.
